# Lead Magnet PDF Guide Templates

This document outlines the structure and content for all 10 lead magnet PDF guides. These PDFs should be created using tools like Canva, Adobe InDesign, or Figma, then uploaded to Uploadthing.

## Template Design Standards

All PDFs should follow the Huts brand guide:
- **Colors:** Black & White, 95% B&W, 5% accent color (only where necessary)
- **Font:** Google Sans
- **Format:** A4 (210mm x 297mm) or Letter (8.5" x 11")
- **Pages:** 30-50 pages per guide
- **File Size:** Optimized < 5MB
- **Include:** Logo, cover page, table of contents, footer with huts.zw

## PDF Creation Workflow

1. **Design in Figma/Canva:**
   - Use Huts brand colors (black/white primary)
   - Consistent header/footer (logo, page numbers, huts.zw)
   - Professional layouts with breathing room

2. **Export:**
   - PDF/X-4 format (print-ready)
   - Compress images to keep under 5MB
   - Test download speed

3. **Upload to Uploadthing:**
   - Use descriptive filenames
   - Tag with lead magnet slug
   - Set expiration (optional)
   - Get public CDN URL

4. **Update Database:**
   - Paste URL into `lead_magnets` table `file_url` column
   - Test download from landing page

5. **Track Downloads:**
   - Monitor via analytics dashboard
   - Iterate content based on feedback

## Content Guidelines

- Use Zimbabwe data (Harare, Bulawayo, Victoria Falls focus)
- Include practical checklists & worksheets
- Reference Huts platform features where relevant
- Use clear headings, bullet points
- Include contact info on last page
- Add CTA to platform at the end
- All prices in both ZWL and USD where applicable
