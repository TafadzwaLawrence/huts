# Favicons & Brand Assets

This folder contains all favicon and design asset files.

## Favicon Files
- Multiple favicon formats and sizes (16x16, 32x32, 192x192, 512x512)
- Apple touch icon for iOS
- Web manifest configuration

## Design Assets
- Brand design files (SVG, PNG)
- Used for OG images, social sharing, and branding

## Usage
These files are referenced in:
- `app/layout.tsx` — favicon and apple-touch-icon meta tags
- `public/site.webmanifest` — web app manifest
